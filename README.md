# Consumer Shopping Behavior Analysis

An end-to-end Data Analytics project that analyzes customer shopping behavior using Python, SQL, Power BI, and a Web-Based Dashboard built with HTML, CSS, and JavaScript.

## Project Overview

This project explores customer purchasing patterns, product preferences, seasonal trends, payment methods, and customer demographics. The goal is to generate actionable business insights and present them through interactive dashboards.
